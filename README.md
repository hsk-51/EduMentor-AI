# EduMentor AI

EduMentor AI is an AI-powered study assistant developed as a capstone project supporting **United Nations Sustainable Development Goal (SDG) 4: Quality Education**.

## Features

- AI Study Assistant
- Notes Summarizer
- Quiz Generator
- Responsive Design
- Modern UI
- Gemini AI Integration (Coming Soon)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Gemini API
- GitHub Pages
